# House Staging Application

A modern web application for house staging, built with **Next.js** (App Router) and **TypeScript**.  

**Live Demo:** [https://house-staging.vercel.app](https://house-staging.vercel.app)

## ✨ Features

- Built with the latest **Next.js 15+** using the App Router
- Fully typed with **TypeScript**
- Optimized font loading with **Geist** (Vercel’s font family)
- Modern styling with **Tailwind CSS** (or custom CSS — extend as needed)
- Ready for production deployment on **Vercel**
- Fast development experience with hot reloading

This project serves as the foundation for a house staging platform — helping users visualize, manage, and present staged properties for real estate marketing.

## 🛠 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: CSS / Tailwind (configurable)
- **Font Optimization**: `next/font`
- **Deployment**: Vercel (recommended)
